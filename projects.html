Projects

WheelDeal

Developed a sophisticated car selling application for iOS using Swift, featuring both light and dark themes for an enhanced user experience. The app showcases multiple cars, each accompanied by detailed information stored in a robust database. Additionally, the application includes a dynamic trivia quiz about cars, which fetches questions and information from an external API, providing users with an engaging and informative experience. This project demonstrates proficiency in Swift programming, UI/UX design, database management, and API integration.
